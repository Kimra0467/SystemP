**운영체제**  
  컴퓨터의 하드웨어 자원을 운영 관리
  프로그램을 실행할 수 있는 환경 제공

**커널(kernel)**  
 운영체제의 핵심으로 하드웨어 운영 및 관리  

**시스템 호출(system call)**  
 커널이 제공한느 서비스에 대한 프로그램이 인터페이스 역할  

**쉘(shell)**  
  사용자와 운영체제 사이의 인터페이스  
  사용자로부터 명령어를 입력받아 해석하여 수행해주는 명령어 해석기
